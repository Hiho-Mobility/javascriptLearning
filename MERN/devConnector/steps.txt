steps to a full stack application : 

1. npm init
2. change the package.json file and add the following to scripts : 
  {
    "start": "node server",
    "server": "nodemon server"
  } 
3. Create server.js file
4. do basic setup
5. create config folder with default.json to have ur mongodb url saved
6. create db.js within the same folder to connect to the db
7. create model for db (create a models folder)
8. create routes and set up
9. app.use(express.json({ extended: false })); put it in server.js to allow json data passing
10.setup user route.. and get verification
11.bunch of verification and authorization setup --> token, middleware, etc
